# terraform-lsp

Terraform language server for Claude Code, providing HCL code intelligence and diagnostics.

## Supported Extensions
`.tf`, `.tfvars`

## Installation

Install terraform-ls using your package manager:

```bash
# macOS
brew install hashicorp/tap/terraform-ls

# Linux / Windows
# Download from the releases page below
```

Make sure `terraform-ls` is in your PATH.
